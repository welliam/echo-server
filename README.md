# http-server
HTTP server for Python 401


# Instructions
Clone this repo, start create new virtual env, install required plugins using:
```
pip install .
```

Start server
```
with venv activated, go to src folder:
python server.py
```

Connect to server
```
in web browser:
go to localhost:5000

```

To run tests, type in command line:
```
pip install .[test]
py.test or tox
```
